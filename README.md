看类型。循环折扣码（recurring，如 `BWHCGLUKKB`）续费自动应用同等折扣；一次性限时码（如 `BWH2020BF`）只在首次下单时打折，续费按原价。

**Q2：优惠码失效了怎么办？**
搬瓦工的码会不定期失效，这是常态。失效后有两个选择：一是直接原价下单（套餐本身性价比已经不差）；二是蹲双十一、黑五的大促码，那段时间通常会放出新的全场折扣码。

**Q3：一个订单能用多个优惠码吗？**
不能。一个订单只能用一个优惠码，建议优先选循环折扣码，长期更划算。

**Q4：老用户能用优惠码吗？**
`BWHCGLUKKB` 这类循环码支持新老用户。但部分限时码会限定新用户首单，下单前看清规则。

**Q5：CN2 GIA-E 和普通 CN2 GT 有什么区别？**
CN2 GIA-E（GT-E）走的是电信 CN2 GIA 优质线路，国内访问延迟低、晚高峰丢包少；CN2 GT 是 CN2 的中端线路，价格便宜但晚高峰容易波动。预算允许就上 GIA-E。

**Q6：香港 CN2 GIA 一直缺货怎么办？**
香港套餐常年库存紧张，建议订阅 stock.bwg.net 的库存通知，或者加入搬瓦工的 Telegram 群组 @BandwagonHostNews，补货后第一时间抢。
